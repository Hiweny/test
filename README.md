# Meteor Shower 流星雨动态背景

唯美的流星雨动态背景页面，纯 Canvas 实现，完美复刻小红书热门视频效果。

## 预览

直接在浏览器中打开 `index.html` 即可查看效果。

## 特性

- **三层 Canvas 架构**：背景层（星空/月亮/地面）、流星层、前景层（萤火虫）
- **蓝紫色调流星**：带光晕头部和渐隐尾迹，模拟真实流星进入大气层的燃烧效果
- **弯月**：右上角暖黄色弯月，带环形山纹理和柔和光晕
- **闪烁星空**：数百颗星星 + 十字星芒效果 + 星座连线
- **深绿植被剪影**：地面草坡 + 白色小花 + 树木剪影
- **萤火虫粒子**：地面附近飘浮的发光粒子，带有呼吸灯效果
- **完美自适应**：支持桌面端和手机竖屏，自动适配屏幕尺寸

## 技术实现

- 纯 HTML/CSS/JavaScript，零依赖
- Canvas 2D 渲染
- 粒子系统实现流星雨
- requestAnimationFrame 实现流畅动画
- 设备像素比适配高清屏

## 使用方式

```bash
# 直接打开
open index.html

# 或使用任意 HTTP 服务器
python3 -m http.server 8080
```
